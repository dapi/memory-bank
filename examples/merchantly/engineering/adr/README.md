# Architecture Decision Records (ADR)

Этот каталог содержит записи архитектурных решений (ADR) для проекта Merchantly.

## Что такое ADR?

ADR — это документ, фиксирующий важное архитектурное решение вместе с контекстом и последствиями. Это помогает:
- Понять **почему** было принято решение
- Сохранить знания для новых членов команды
- Избежать повторного обсуждения уже принятых решений
- AI-агентам следовать установленным паттернам

## Статусы

- **Предложено** — решение обсуждается
- **Принято** — решение принято и должно соблюдаться
- **Устарело** — заменено другим ADR
- **Отклонено** — решение отклонено после обсуждения

- ...
